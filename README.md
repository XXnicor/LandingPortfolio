# Landing Portfolio - Frontend Developer

Landing page profissional construída com Next.js 14, React, TypeScript e TailwindCSS.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build de produção

```bash
npm run build
npm start
```

## Checks

```bash
npm run lint
```

> Neste template o script `lint` executa `tsc --noEmit` (typecheck), reduzindo dependências legadas do ecossistema ESLint que geravam warnings deprecados no install.
